# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-05-10
### Added
- **Docker Support**: Official Debian Slim based Docker image and docker-compose template.
- **English Localization**: Entire codebase, comments, and documentation translated to English.
- **Flexible Telegram Intervals**: Configurable intermediate report frequency.
- **Professional Icons**: New technical emoji set for sensors.
- **GitHub Actions**: Added ShellCheck CI workflow.

## [1.1.0] - 2026-05-09
### Added
- **Multi-Sensor Support**: Unified tracking for Intel RAPL (Package/Core/DRAM/iGPU).
- **NVIDIA Integration**: Real-time power monitoring via `nvidia-smi`.
- **Disk Power Estimation**: Hybrid model for HDDs and SSDs.
- **Power Source Tracking**: Automatic AC/Battery detection and battery percentage.

## [1.0.0] - 2026-05-08
### Added
- Initial release with basic Intel RAPL monitoring and daily Telegram reports.
